Acerca de este sitio

Este sitio web ha sido desarrollado como parte del curso de Programación IV, con el propósito de aplicar continuamente los conocimientos teóricos y prácticos mediante la construcción de un sitio educativo funcional, que a la vez servirá como repositorio de los temas estudiados en clase.

El sitio está organizado en diferentes temas como: fundamentos de Internet y la Web, HTML, CSS y conceptos básicos de backend, incluyendo explicaciones, ejemplos y fragmentos de código.

Este sitio ha sido desarrollado utilizando HTML5 para la estructura, CSS para los estilos y el framework Bootstrap para el diseño responsive, además de JavaScript para la interacción dinámica.

Estesitio fue desarrollado por Nicolas Zárate Hernández.
